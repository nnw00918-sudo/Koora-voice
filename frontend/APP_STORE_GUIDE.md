# 📱 دليل رفع تطبيق "صوت الكورة" على App Store

## المتطلبات الأساسية

### 1. حساب Apple Developer
- تأكد من وجود حساب Apple Developer Program ($99/سنة)
- قم بتسجيل الدخول على [App Store Connect](https://appstoreconnect.apple.com)

### 2. برنامج Xcode
- قم بتثبيت Xcode من Mac App Store (يتطلب macOS)
- تأكد من تثبيت Xcode Command Line Tools

---

## خطوات الرفع

### الخطوة 1: تحميل المشروع على Mac

1. انسخ مجلد `/app/frontend` كاملاً إلى جهاز Mac
2. أو استخدم Git لسحب المشروع

```bash
# على Mac
cd ~/Desktop
# انسخ المشروع من السيرفر
```

### الخطوة 2: تثبيت المتطلبات

```bash
cd frontend
yarn install
npx cap sync ios
```

### الخطوة 3: فتح المشروع في Xcode

```bash
npx cap open ios
```

أو افتح الملف يدوياً:
`frontend/ios/App/App.xcworkspace`

### الخطوة 4: إعداد التوقيع (Signing)

1. في Xcode، اختر المشروع "App" من الشريط الجانبي
2. اذهب إلى تبويب "Signing & Capabilities"
3. اختر Team (حساب Apple Developer الخاص بك)
4. تأكد من تفعيل "Automatically manage signing"
5. Bundle Identifier: `com.kooravoice.app`

### الخطوة 5: إعداد الأيقونات (مهم!)

1. افتح `Assets.xcassets` > `AppIcon`
2. أضف أيقونة 1024x1024 بكسل
3. Xcode سيقوم بإنشاء الأحجام المختلفة تلقائياً

**ملاحظة:** تم نسخ `logo512.png` كأيقونة مؤقتة. يُفضل استخدام أيقونة مخصصة 1024x1024.

### الخطوة 6: بناء التطبيق للنشر

1. اختر "Any iOS Device" من قائمة الأجهزة
2. اذهب إلى Product > Archive
3. انتظر حتى يكتمل البناء

### الخطوة 7: رفع التطبيق

1. بعد اكتمال Archive، ستفتح نافذة Organizer
2. اختر Archive واضغط "Distribute App"
3. اختر "App Store Connect"
4. اتبع الخطوات لرفع التطبيق
